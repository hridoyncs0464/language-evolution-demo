# Language Evolution Demo

A responsive, academic-style static website built using pure HTML5 and CSS3.

This project demonstrates a clean, mobile-first approach to creating multi-page informational websites without relying on frameworks, external libraries, or JavaScript.

---

## Overview

The website explores themes related to language development, cultural communication, and the evolution of ideas through a simple and accessible academic-style design.

The project includes:

* Home Page
* Living Languages Page
* Evolution Page

---

## Features

* Mobile-first responsive design
* Multi-page website structure
* Semantic HTML5 markup
* Pure CSS styling
* Clean typography and readable layout
* No frameworks or dependencies
* GitHub Pages compatible

---

## Technologies Used

* HTML5
* CSS3
